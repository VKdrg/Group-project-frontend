# Projet scolaire - Conception d'une application web

## Objectif 
Concevoir une application desktop et mobile locale qui permette de suivre ses cadidatures. Travailler en binôme pour la conception, le maquettage et le développement d'une app web, en lien avec un groupe de back-end pour le développement d'une API qui contienne les détails.

## Technos utiliées
Front :
- HTML
- CSS
- JavaScript
- Git / GitHub

## Outils de Qualité de Code
Utilisation d'utilitaires de contrôle de qualité de code :
- HTML: HTML Hint
- CSS : Stylelint
- JS : ESLint

## Étapes à réaliser
1. Concevoir la charte graphique
2. Choisir les fonctionalités de l'app
3. Maquetter l'interface utilisateur
4. Développer l'interface front-end
5. Lier l'API créée par le groupe de back-en
